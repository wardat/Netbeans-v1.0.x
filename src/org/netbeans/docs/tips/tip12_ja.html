メインウィンドウのメニューとツールバーは、「汎用オプション」ウィンドウでカスタマイズできます。使用するアクションを変更するには「アクション」、メインウィンドウのメニューバーをカスタマイズするには「メニュー」、ツールバーの構成を変更するには「ツールバー」を使用します。クリップボード (コピーとペースト) を使用してツールバーとメニューを好みのデザインに構成し直すこともできます。

ツールバーのグリップ部分にカーソルをあわせてツールバーをつかみ、ドラッグして配置を変えます。ツールバー領域の空白部分を右クリックすると、コンテキストメニューが表示されます。ここで、現在の構成のツールバーを追加・削除し、新しい構成を作成することができます。

また、独自のメニューやツールバー項目を作ることもできます。main メソッドを持つ Java クラスを作成し、そのメソッドに実行したい内容を指定します。次にそのクラスが内部実行を使用するように設定します。具体的には、ソースファイルのプロパティシートの「実行」タブで 「実行方法」プロパティに「内部実行」 を指定選択します。最後にそのクラスをコピーして、「汎用オプション」ウィンドウの 「メニュー」ノードまたは「ツールバー」ノードにペーストします。
