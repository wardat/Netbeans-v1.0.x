実行の種類は、各ファイルごとに外部実行と内部実行のどちらにでも設定できます。外部実行は新しい Java VM で行われる実行です。内部実行は IDE 自体を変更したい場合に便利です。このほかにもアプレット実行や、インストールされているモジュールから提供される実行も使用できます。

オブジェクトに使用する実行の種類を変えるには、まずエクスプローラでオブジェクトのアイコンをクリックします。次に「プロパティ」ウィンドウ中のプロパティシートの「実行」タブをクリックします。「実行」タブ上で「実行方法」プロパティを選択して、表示される下向きの矢印をクリックします。表示されたコンボボックスから実行の種類を選択します。

実行の種類は「プロジェクト設定」ウィンドウでも構成できます。メインメニューの「プロジェクト」から「設定...」を選択し、「プロジェクト設定」ウィンドウで「実行の種類」を選択してください。
