Java で必要なパッケージやクラスにアクセスするには、CLASSPATH に正しいディレクトリ階層とアーカイブを含める必要がありますが、これと同様に IDE のファイルシステム (Filesystems)中で新しいファイルをマウントする位置も大変重要です。

デフォルトのパッケージが含まれているディレクトリは必ずマウントしてください。たとえば、パッケージ com.yoyodyne.spreadsheet 中に自分で作成したクラスがあり、そのクラスがディレクトリ C:\JavaDevel\source\com\yoyodyne\spreadsheet\ に含まれている場合、必ず C:\JavaDevel\source\ ディレクトリをマウントしてください。

同様に、JAR ファイルを使用する場合は、Java パッケージと JAR ファイル中のクラスの位置が合致していなければなりません。これらの設定は Java では必須条件なので、IDE でも習慣付けることをお勧めします。
