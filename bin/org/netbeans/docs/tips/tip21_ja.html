Java クラスのコンパイルには、どの Java コンパイラを使用してもかまいません。IDE が使用できるコンパイラを新たに定義するには、メインウィンドウのメニュー「プロジェクト」 から「設定...」を選択し、「コンパイラの種類」ノードを展開します。

種類を示すサブノードを右クリックして、コンテキストメニューから「新規 "コンパイラ名" サービス」を選択します ("コンパイラ名" には各コンパイラの名前が表示されます)。種類のノードの下に新しいコンパイラ用のサブノードが表示されます。このノードのプロパティシートで新しいコンパイラを設定することができます。

クラス用のプロパティシートには「実行」という名前のタブがあり、その中に「コンパイラ」というプロパティがあります。このプロパティを新たに追加したコンパイラに設定してください。

デバッガと実行も同じ方法で設定できます。デバッガには「デバッガの種類」を、実行には「実行の種類」を「プロジェクトの設定」ウィンドウで選択してください。
