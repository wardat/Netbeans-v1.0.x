Forte for Java (Community Edition バージョン 1.0) へようこそ

Forte for Java (Community Edition) は、使用中の開発スタイルにすぐに適用できる、完全にオープンなシステムです。

この製品の主要な機能の概要を短時間で身に付けるには、『Forte(TM) for Java(TM) (Community Edition) クイックスタートガイド』を参照し、例題プログラムを実行してください。このマニュアルは、「ヘルプ」メニューの「クイックスタート」で表示されます。

注 - 現段階では、上記の方法で表示できるマニュアルは英語版のみです。日本語版については、インストール先の docs ディレクトリにある PDF ファイルを参照してください。
